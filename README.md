# persistent_login

This is a simple Flutter application demonstrating persistent login using the Clean Architecture pattern.

## Getting Started

- This project uses Flutter SDK version **3.32.6**

## Architecture

- This project follows the principles of Clean Architecture, separating concerns into data, domain,
  and presentation layers.

## Built with

- BLoC
- GetIt
- GoRouter
- Flutter Secure Storage
- Http
- Equatable

## Mockup API

- This project uses a mock API server via [Mockoon](https://mockoon.com/).  
  The configuration file is provided at `mockoon/login.json`.

### To run the mock API:
1. Open Mockoon
2. Import the `login.json` file
3. Start the server on the designated port
